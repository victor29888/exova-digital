# Exova Digital

Site vitrine premium pour **Exova Digital**, agence digitale accompagnant les PME françaises.

## Stack

- [Next.js](https://nextjs.org) (App Router)
- TypeScript
- Tailwind CSS

## Démarrage

```bash
npm install
npm run dev
```

Ouvrir [http://localhost:3000](http://localhost:3000).

## Sections

- Hero
- Services (sites web, IA, publicité)
- Process (méthode en 4 étapes)
- Pourquoi nous
- Tarifs
- FAQ
- Contact

## Build production

```bash
npm run build
npm start
```
